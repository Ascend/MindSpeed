# Security Statement

The security hardening measures listed in this document are basic hardening recommendations. Users should re-evaluate the network security hardening measures for the entire system based on their own business needs. Users should perform relevant configurations in accordance with their organization's security policies, including but not limited to software versions, system hardening requirements, security configurations (data storage, debug interfaces, runtime environments, etc.), permission configurations, and firewall settings. When necessary, users may refer to industry best practices for hardening and advice from security experts.
