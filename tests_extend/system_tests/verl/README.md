# VERL System Tests

该目录汇总 MindSpeed 对 VERL 的系统级测试脚本。每个模型一个子目录，README 只描述该模型的训练环境与步骤，避免脚本增加后混淆。

## 目录结构

- `qwen3_30b/`：Qwen3-30B-A3B 训练脚本与说明
- `qwen3_8b/`：Qwen3-8B 训练脚本与说明（待补充脚本）

## 使用方式

1. 进入对应模型目录查看 `README.md`。
2. 按说明准备权重、数据与环境。
3. 在 `verl` 代码仓内执行脚本。

## 新增脚本约定

- 每个模型一个子目录。
- 目录内至少包含 `README.md` 与 `run_*.sh` 或已有脚本命名。
